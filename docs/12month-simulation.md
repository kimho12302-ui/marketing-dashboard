# 🧪 12개월 마케터 시뮬레이션

> 2년차 마케터 "김마케"가 PPMI 대시보드를 1년간 사용하며 느낄 불편점을 월별로 시뮬레이션

---

## Month 1 (1월): 온보딩 — "이게 뭐지?"

### 시나리오
김마케 입사 첫 달. 대시보드를 처음 본다.

### 불편점
1. **어디서부터 봐야 하는지 모름** — 탭이 많은데 가이드가 없음
2. **용어 설명 없음** — MER, AOV, CAC가 뭔지 모르는 신입도 있을 수 있음
3. **데이터 입력을 어떻게 하는지 모름** — 매뉴얼이 대시보드 안에 없음
4. **"이 숫자가 좋은 건지 나쁜 건지"** 판단 기준 없음

### 개선
- [x] 일일 입력 가이드 (이미 구현: DailyInputGuide)
- [ ] **온보딩 투어**: 처음 접속 시 주요 기능 설명 오버레이
- [ ] **용어 사전 툴팁**: KPI 카드에 (?) 아이콘 → 호버 시 설명
- [ ] **데이터 입력 매뉴얼 페이지**: Settings 내 또는 별도 /guide 페이지
- [ ] **벤치마크 표시**: 업종 평균 ROAS, CTR 등 참고선

## Month 2 (2월): 루틴 형성 — "매일 입력하기 귀찮다"

### 시나리오
매일 쿠팡 광고비 다운→업로드, GFA 수기 입력 반복.

### 불편점
1. **쿠팡 파일 업로드가 번거로움** — 로그인→다운로드→업로드 3단계
2. **GFA 금액 매번 수동 입력** — 실수 가능
3. **입력 빠뜨려도 알림 없음** — 3일 지나서야 데이터 구멍 발견
4. **어제 입력했는지 확인하기 어려움** — 입력 이력 없음

### 개선
- [ ] **입력 누락 알림**: 전날 데이터 미입력 시 대시보드 상단 경고
- [ ] **입력 이력 로그**: Settings에 최근 입력 기록 테이블
- [ ] **드래그앤드롭 업로드**: 파일을 Overview에 바로 드롭
- [ ] **빠른 입력 모달**: Overview에서 바로 "오늘 광고비 입력" 버튼

## Month 3 (3월): 보고서 시즌 — "팀장님한테 뭘 보여주지?"

### 시나리오
분기 보고. 대시보드 캡처해서 PPT 만들어야 함.

### 불편점
1. **보고서 내보내기 없음** — 스크린샷 찍어서 파워포인트에 붙이는 중
2. **특정 기간 요약이 안 됨** — "3월 성과 한눈에"가 없음
3. **전월 대비 한방에 안 보임** — Overview는 기간 하나만 표시
4. **브랜드별 보고서 분리 어려움** — 너티/아이언펫 각각 캡처

### 개선
- [ ] **PDF/이미지 보고서 내보내기**: 원클릭 리포트 다운로드
- [ ] **기간 요약 카드**: 월간/주간 하이라이트 자동 생성
- [ ] **비교 모드**: 이번 달 vs 지난달 나란히 보기
- [ ] **브랜드별 리포트**: 브랜드 선택 → 해당 브랜드 전체 KPI 한 페이지

## Month 4 (4월): 광고 최적화 — "어떤 광고를 꺼야 하지?"

### 시나리오
광고 예산 조정 필요. 소재별 성과를 비교하려는데...

### 불편점
1. **소재 A/B 비교 기능 없음** — 나란히 비교 못함
2. **기간별 소재 성과 변화 못 봄** — 이번 주 vs 지난주 소재 성과
3. **예산 시뮬레이션 없음** — "Meta 예산 50만 늘리면 매출 얼마 오를까?"
4. **소재 태깅 없음** — 영상/이미지/카러셀 구분이 name으로만

### 개선
- [ ] **소재 비교 뷰**: 2-3개 소재 나란히 비교
- [ ] **소재 성과 트렌드**: 특정 소재의 일별 ROAS 변화
- [ ] **예산 시뮬레이터**: 채널별 예산 조정 → 예상 ROAS/매출 계산
- [ ] **소재 태그**: 자동 분류 (영상/이미지/카러셀) + 수동 태그

---

## Month 5 (5월): 콘텐츠 연결 — "블로그 효과가 있는 건지 모르겠는데"

### 시나리오
블로그 5개 운영 중인데 어떤 글이 매출에 기여하는지 모름.

### 불편점
1. **블로그→매출 연결 없음** — Content 탭이 비어있음
2. **인스타 성과 못 봄** — API 연결 안 됨
3. **콘텐츠 ROI 계산 불가** — 글 하나에 투입된 비용 vs 유입 매출
4. **어떤 콘텐츠를 더 만들어야 하는지 기준 없음**

### 개선
- [ ] **블로그 크론 연동**: 5개 블로그 일일 방문자/댓글 자동 수집
- [ ] **인스타 API 연동**: 게시물별 도달/참여/저장
- [ ] **콘텐츠→유입→구매 추적**: UTM 기반 콘텐츠 기여도
- [ ] **콘텐츠 성과 대시보드**: 게시물별 유입수, 전환수, ROI

---

## Month 6 (6월): 상반기 회고 — "6개월 치 트렌드가 한눈에 안 보여"

### 시나리오
상반기 성과 정리. 6개월 데이터를 한번에 보려는데...

### 불편점
1. **장기 트렌드 보기 어려움** — 30일 뷰가 기본, 6개월은 너무 빽빽
2. **시즌별 비교 안 됨** — 작년 동기 대비가 없음 (첫 해니까)
3. **누적 성과 없음** — YTD(Year-to-Date) 매출, 누적 광고비
4. **목표 달성률 추이 없음** — 월별 목표 vs 실적 그래프

### 개선
- [ ] **월별 요약 뷰**: 월 단위로 KPI 추이 (라인/바 차트)
- [ ] **YTD 누적 카드**: 연초~현재까지 총 매출, 총 광고비, 평균 ROAS
- [ ] **목표 달성률 추이 차트**: 월별 목표 대비 실적 %
- [ ] **하이라이트 자동 생성**: "6월 최고 매출일", "ROAS 최고/최저 주" 등

## Month 7 (7월): 채널 확장 — "새 채널 추가하려는데"

### 시나리오
밸런스랩 런칭, 새 광고 채널 테스트 (틱톡 등)

### 불편점
1. **새 채널 추가 절차 복잡** — DB 스키마, API 연동, UI 모두 수동
2. **채널 간 비교가 어려움** — 채널마다 지표 기준이 다름
3. **밸런스랩 데이터가 별도** — 브랜드 추가 시 대시보드 설정 변경 필요
4. **테스트 채널 성과 따로 관리 안 됨**

### 개선
- [ ] **채널 설정 UI**: Settings에서 새 채널 추가/제거
- [ ] **통합 채널 비교**: 채널별 CPM, CPC, CTR, ROAS 나란히
- [ ] **브랜드 설정 UI**: 새 브랜드 추가 (색상, 이름)
- [ ] **테스트/메인 채널 구분**: 테스트 채널 별도 표시

## Month 8 (8월): 자동화 욕구 — "반복 작업 줄이고 싶다"

### 시나리오
매일 같은 루틴. 더 자동화하고 싶음.

### 불편점
1. **쿠팡 광고비 여전히 수동** — 자동화 방법 없나?
2. **주간 보고서 매번 수동 작성** — 자동 생성 원함
3. **이상치 감지가 대시보드 열어야만 보임** — 텔레그램 알림 원함
4. **데이터 검증이 수동** — 이상한 데이터 들어와도 모름

### 개선
- [ ] **쿠팡 광고 자동 수집**: 브라우저 자동화 (Playwright)
- [ ] **주간 보고서 자동 생성 크론**: 매주 금요일 자동 리포트
- [ ] **텔레그램 알림 연동**: 이상치 감지 시 즉시 알림
- [ ] **데이터 검증 룰**: 매출 0원, ROAS 100+ 등 이상값 자동 필터

## Month 9 (9월): 깊이 분석 — "왜 매출이 떨어졌는지 알고 싶다"

### 시나리오
9월 매출 하락. 원인을 찾으려는데...

### 불편점
1. **드릴다운이 부족** — 매출 하락 → 어떤 브랜드? → 어떤 채널? → 어떤 제품?
2. **원인 자동 분석 없음** — 사람이 일일이 탭 돌아다녀야 함
3. **외부 요인 반영 안 됨** — 시즌, 경쟁사, 이벤트 등
4. **코호트 분석 없음** — 신규 vs 재구매 고객 구분

### 개선
- [ ] **자동 원인 분석**: "매출 -20% → 너티 -30% → 카페24 -40% → 사료 카테고리 주문 감소"
- [ ] **이벤트 마커**: 캘린더에 프로모션, 시즌 이벤트 표시 → 차트에 마커
- [ ] **코호트 분석**: 신규/재구매 분리 (퍼널에 재구매 데이터 필요)
- [ ] **경쟁사 영향 메모**: 날짜별 메모 기능 (경쟁사 할인, 시즌 등)

## Month 10 (10월): 예측 — "다음 달 매출 얼마나 될까?"

### 시나리오
내년 예산 계획. 예측이 필요한데...

### 불편점
1. **매출 예측 없음** — 과거 트렌드 기반 예측선
2. **시나리오 분석 없음** — "광고비 20% 늘리면?"
3. **시즌 보정 없음** — 11월(블프), 1월(설) 시즌성 반영
4. **재고 연동 없음** — 재고 소진 속도 vs 판매 속도

### 개선
- [ ] **매출 예측 차트**: 이동평균 + 트렌드 라인
- [ ] **시나리오 시뮬레이터**: 광고비/채널 조정 → 예상 결과
- [ ] **시즌 달력**: 주요 시즌 + 이벤트 + 예상 매출 효과
- [ ] **재고 알림**: Ecount ERP 연동 → 재고 부족 경고

## Month 11 (11월): 블프/대형 이벤트 — "실시간으로 보고 싶다"

### 시나리오
블랙프라이데이 대규모 프로모션. 실시간 현황 필요.

### 불편점
1. **실시간 데이터 없음** — 크론이 1일 1회라 오늘 현황 못 봄
2. **이벤트 대시보드 없음** — 특정 프로모션 기간 성과만 보기
3. **재고 실시간 확인 불가** — 인기 상품 품절 모니터링
4. **팀 공유 어려움** — 대시보드 URL 공유 시 로그인 문제

### 개선
- [ ] **실시간 모드**: 5분 간격 데이터 갱신 (이벤트 기간 한정)
- [ ] **이벤트 대시보드**: 프로모션 기간 설정 → 해당 기간 전용 뷰
- [ ] **재고 모니터링**: ERP 연동 실시간 재고 + 판매 속도
- [ ] **공유 링크**: 비로그인 읽기 전용 뷰 (기간 한정)

## Month 12 (12월): 연말 회고 — "올해 뭘 잘했고 뭘 못했지?"

### 시나리오
연간 성과 정리. CEO에게 보고.

### 불편점
1. **연간 리포트 없음** — 12개월 전체 요약
2. **채널별 ROI 랭킹 없음** — 어떤 채널이 가장 효율적이었나
3. **제품별 수익성 랭킹 없음** — 어떤 제품이 가장 돈을 벌었나
4. **성장률 자동 계산 없음** — MoM, QoQ 추이

### 개선
- [ ] **연간 리포트 페이지**: 12개월 KPI 요약 + 하이라이트
- [ ] **채널 ROI 랭킹**: 연간 채널별 투자 대비 수익
- [ ] **제품 수익성 분석**: 매출 - 원가 - 광고비 = 순이익 by 제품
- [ ] **성장 지표**: MoM 매출 성장률, 누적 고객 수, LTV 추정
