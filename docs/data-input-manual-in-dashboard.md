# 대시보드 내 데이터 입력 가이드 페이지 설계

Settings > 📋 일일 입력 탭에 이미 DailyInputGuide 컴포넌트가 있음.
추가로 /guide 페이지를 만들어 매뉴얼 접근성 높이기.

## 구현 완료 목록:
1. ✅ 일일 입력 가이드 (DailyInputGuide)
2. ✅ 데이터 현황 모니터링 (DataStatusCard)  
3. ✅ 입력 누락 알림 (MissingDataAlert)
4. ✅ 이상치 알림 (AnomalyBanner)
5. ✅ 용어 사전 툴팁 (KPI 카드 ? 아이콘)
6. ✅ 목표 설정 탭 (TargetsTab)
7. ✅ 크리에이티브 퍼널 테이블
8. ✅ 데이터 입력 매뉴얼 문서
